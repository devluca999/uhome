# Post-MVP Roadmap — uhome

## Overview

This document outlines future features and enhancements planned for uhome after MVP launch. These features are documented for planning purposes but are not commitments—priorities may shift based on user feedback and business needs.

## Map & Location Features

### Interactive Property + Home Map
- Visual map view of all properties
- Property locations marked on map
- Click to navigate to property details
- Filter properties by location

### Tenant & Landlord Shared Map View
- Tenants see their property location
- Landlords see all property locations
- Shared context for maintenance and visits
- Location-based features

### One-Tap Navigation Workflows
- "Go home" button for tenants (opens directions)
- "Go to property" button for landlords
- Integration with mapping apps (Google Maps, Apple Maps)
- Quick access to property locations

### Potential Brand Collaborations
- Uber/Lyft integration for property visits
- One-tap ride booking to properties
- Branded workflows for property management
