# Design Direction — haume

Design principles:
- Calm
- Expressive minimalism
- Neutral power balance
- Ambient, glassmorphic surfaces

Visual language:
- Warm whites and soft grays
- Single accent hue (sage or slate)
- Status via glow and tint
- Subtle motion (150–250ms)

Avoid:
- Flat, lifeless cards
- Loud gradients
- Over-animation
- Playful or cartoonish elements

Inspiration:
- Notion
- iOS system UI
- Linear
